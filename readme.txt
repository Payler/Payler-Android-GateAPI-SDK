Payler GateAPI SDK
==================

Проект для Android Studio.

1. Запустить Android Studio.
2. Выбрать Open Project.
3. Выбрать каталог android-gateapi-sdk

Каталог lib содержит непосредственно библиотеку.
Каталог app содержит демонстранионное приложение.

Для запуска демонстрационного приложения достаточно открыть проект в Android
Studio, выбрать Build -> Make Project, Run -> Run 'app'

Демонстрационное приложение позволяет сначала выполнить одностадийный платёж
на сумму 10 рублей, а потом сделать частичный возврат на 1 рубль.

Ознакомьтесь, пожалуйста, с документацией по Gate API
https://drive.google.com/open?id=0B5A1Tl-BiPnSSk1aOWRWYjMyWWM.

Для выполнения любой операции через API или SDK нужно иметь ключ доступа
(merchant key) и для некоторых операций платёжный пароль.  Для получения этих
данных, а также номеров тестовых карт, обратитесь, пожалуйста, в службу
поддержки 24@payler.com
